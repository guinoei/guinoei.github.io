# Guilherme Schilling — Portfolio

An RPG-styled developer portfolio. Inspired by the UI language of Breath of the Wild, Dark Souls, and Elden Ring — dark atmosphere, glassmorphism panels, gold accents, and game-inventory aesthetics.

Built with pure HTML, CSS, and JavaScript.

**[guinoei.com](https://guinoei.com)**

---

## Features

- **Cinematic landing screen** — layered mountain silhouettes, particles field, fog and vignette overlays
- **Game modal** — tabbed panel with Profile, Projects, Creative, Skills, and Contact sections
- **Quest log** — split-panel project browser with per-project descriptions and tech stacks
- **Commission log** — service cards for creative work (video editing, branding, operations)
- **Settings modal** — language selector with EN / PT-BR / ES support
- **Full i18n** — all visible text translated; project titles, dates, and descriptions localized per language; preference saved to localStorage
- **Responsive** — mobile sheet slides up from the bottom; panels reflow for small screens
